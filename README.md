# Moti - Homebrew Formula

## Installation

### Quick Install
```bash
brew install https://raw.githubusercontent.com/ezedepetris/homebrew-moti/main/moti.rb
```

### Or tap first
```bash
brew tap ezedepetris/moti https://github.com/ezedepetris/homebrew-moti
brew install moti
```

## Updating
```bash
brew upgrade moti
```

## Uninstall
```bash
brew uninstall moti
brew untap ezedepetris/moti
```

---

## About Moti

Moti is a native macOS menu bar widget that delivers motivational quotes straight to your desktop.

Learn more: https://motiapp.com
